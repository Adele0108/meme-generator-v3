表情包生成器 - 运行说明
====================

问题：如果直接双击打开 HTML 文件（使用 file:// 协议），可能会遇到以下问题：
1. 图片转换失败
2. 无法下载图片

解决方法（任选一种）：
--------------------

方法 1：使用 VS Code Live Server（推荐）
1. 在 VS Code 中安装 "Live Server" 扩展
2. 右键点击 meme-generator.html
3. 选择 "Open with Live Server"
4. 浏览器会自动打开网页

方法 2：使用 Python（如果已安装 Python）
1. 在 meme-generator-v3 文件夹中打开命令行/终端
2. 运行命令：python -m http.server 8000
3. 在浏览器访问：http://localhost:8000/meme-generator.html

方法 3：使用 Node.js（如果已安装 Node.js）
1. 在 meme-generator-v3 文件夹中打开命令行/终端
2. 运行命令：npx http-server
3. 按照提示在浏览器访问显示的地址

方法 4：使用其他本地服务器
- 可以使用任何支持静态文件的 HTTP 服务器

注意：
- 使用 HTTP 服务器运行后，所有功能都会正常工作
- 如果使用 file:// 协议，可以右键点击图片选择"另存为"来保存（图片已包含文字）
